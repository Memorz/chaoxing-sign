## ✨Features

- [x] 普通签到
- [x] 拍照签到
- [x] 手势签到
- [x] 位置签到
- [x] 签到码签到
- [x] 二维码签到（10秒变换不影响）
- [x] 多用户凭据储存

## 🛠 部署 

```shell
git clone https://github.com/kuizuo/chaoxing-sign.git
cd chaoxing-sign
pnpm install
```

你需要一个 PostgreSQL 数据库地址，然后将项目根目录下 `.env.example` 文件更改成 `.env` 并填入相关信息。执行

```shell
pnpm run dev
```
